**提示**：本篇博客代码至少需要开启 `-std=c++11` 选项来支持 C++ 11 相关特性， C++ 14 特性的相关代码需要开启 `-std=c++14` 的编译选项。

```bash
$ g++ main.cpp -std=c++11

$ g++ main.cpp -std=c++14
```
