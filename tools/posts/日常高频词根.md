### 1. cent(i) = hundred  百

| 词汇           | 词义                          | 联想                  | 相关                                           |
| -------------- | ----------------------------- | --------------------- | ---------------------------------------------- |
| cent           | 美分                          |                       |                                                |
| centigrade     | 摄氏的                        | centi + grade（级）   | minus 3 degrees centigrade  零下三度           |
| percent        | 百分比                        | per（每）+ cent       | **percentile**  百分率的                       |
| percentage     | 百分率                        | per（每）+ cent + age | What percentage of … ?  …百分之几？            |
| centimeter     | 厘米                          | centi + meter（米）   |                                                |
| **centennial** | a. 百年纪念的</br>n. 百年纪念 | cent + enn（年）+ ial | centenary  一百年</br>centennially  每百年一次 |
